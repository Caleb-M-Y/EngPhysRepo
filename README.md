# EngPhysRepo

This repository was made to be passed down to Engineering Physics majors or any others who have to take physics, math, etc. This repository is for members of Kappa Sigma at UCA and whoever they see fit to use it.

## Repository Structure

Work uploaded here may include lab reports, homework assignments, tests, projects, and more, organized by course.

```
EngPhysRepo/
├── Math/
│   ├── Calculus_I/
│   ├── Calculus_II/
│   ├── Calculus_III/
│   ├── Linear_Algebra/
│   └── Ordinary_Differential_Equations/
│
├── Physics/
│   ├── Physics_I/
│   ├── Physics_II/
│   ├── Physics_III/
│   ├── Electromagnetism_I/
│   └── Electromagnetism_II/
│
└── Upper_Classman_Courses/
    ├── Statics/
    ├── Dynamics/
    ├── Microelectronics/
    ├── Electronics/
    ├── Microcontrollers/
    ├── Robotics_I/
    ├── Robotics_II/
    └── Acoustics/
```

Each course folder contains the following subdirectories:

| Subdirectory  | Contents                        |
|---------------|---------------------------------|
| `Lab_Reports` | Lab reports                     |
| `Homework`    | Homework assignments            |
| `Tests`       | Tests and exams                 |
| `Projects`    | Projects and capstone work      |
