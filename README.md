# EngPhysRepo

This repository was made to be passed down to Engineering Physics majors or any others who have to take physics, math, etc. This repository is for members of Kappa Sigma at UCA and whoever they see fit to use this.

---

## Repository Structure

```
EngPhysRepo/
├── math/
│   ├── calc1/            Calculus I
│   ├── calc2/            Calculus II
│   ├── calc3/            Calculus III (Multivariable)
│   ├── diff-eq/          Differential Equations
│   └── linear-algebra/   Linear Algebra
│
└── physics/
    ├── mechanics/         Classical Mechanics (Physics I)
    ├── electromagnetism/  Electromagnetism (Physics II)
    ├── modern-physics/    Modern Physics
    └── thermodynamics/    Thermodynamics & Statistical Mechanics
```

Each folder has its own `README.md` with course-specific resources and a place to drop notes, cheat sheets, practice problems, and past exams.

## Quick-Reference Resources

|----------|----------------|
| [Paul's Online Math Notes](https://tutorial.math.lamar.edu/) | Calc I–III, Diff Eq — free notes & practice problems |
| [Professor Leonard (YouTube)](https://www.youtube.com/c/ProfessorLeonard) | Full lecture series: Calc I–III & Diff Eq |
| [3Blue1Brown – Calculus](https://www.youtube.com/playlist?list=PLZHQObOWTQDMsr9K-rj53DwVRMYO3t5Yr) | Visual intuition for calculus |
| [3Blue1Brown – Linear Algebra](https://www.youtube.com/playlist?list=PLZHQObOWTQDPD3MizzM2xVFitgF8hE_ab) | Visual intuition for linear algebra |

| [Khan Academy – Math](https://www.khanacademy.org/math) | Video lessons, all levels |
